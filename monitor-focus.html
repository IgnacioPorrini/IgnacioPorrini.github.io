Monitor Focus

App de escritorio para macOS y Windows que oscurece instantáneamente los monitores que no estás usando mediante un atajo de teclado configurable — pensada para reducir la fatiga visual al trabajar con varias pantallas.

Electron JavaScript macOS Windows
Ver repositorio Descargar código fuente

¿Cómo funciona?

Monitor Focus no apaga el monitor a nivel de hardware (eso depende del soporte DDC/CI de cada pantalla y es poco confiable multiplataforma). En cambio, crea una ventana negra, sin bordes y siempre-encima que cubre cada monitor y le aplica una opacidad configurable (95% por defecto) sobre lo que ya está en pantalla. Es instantáneo, reversible y funciona igual en cualquier monitor o GPU.

La capa de oscurecido es click-through: no bloquea el mouse ni el teclado, así que las apps debajo siguen siendo usables — es un filtro visual, no un bloqueo de pantalla.

Por defecto, Shift+1 enfoca el monitor 1 (oscurece los demás), Shift+2 enfoca el monitor 2, y así sucesivamente. Todo es configurable desde Preferencias.

Modos de foco

Elegís uno desde Preferencias

Foco

(Predeterminado) El atajo de un monitor oscurece todos los demás y deja ese encendido. Presionarlo de nuevo sobre el mismo monitor enfocado vuelve a encender todo. Pensado para "quiero concentrarme en el monitor 1 ahora".

Independiente

Cada atajo prende/apaga el oscurecido de ese monitor únicamente, sin tocar los demás. Podés oscurecer varios a la vez si querés.

Guía rápida de instalación

Todavía no hay un instalador precompilado (.dmg / .exe). Por ahora, la app se corre desde el código fuente con Node.js — es rápido y no requiere herramientas extra.
1
Instalá los requisitos: Node.js 18 o superior, en macOS o Windows.
2
Descargá el proyecto con el botón de arriba, o cloná el repositorio: git clone https://github.com/IgnacioPorrini/monitor-focus.git
3
Instalá las dependencias y arrancá la app desde la carpeta del proyecto: npm install
npm start Al arrancar no vas a ver ninguna ventana: la app vive en la bandeja del sistema (Windows) o la barra de menú (macOS). Hacé click en el ícono para ver el menú y abrí Preferencias para configurar atajos, intensidad y modo.
4
(Opcional) Generá tu propio instalador para no depender de Node.js día a día: # En macOS, genera un .dmg
npm run dist:mac

# En Windows, genera un instalador .exe
npm run dist:win Como la app no está firmada con certificado de desarrollador, la primera vez que la abras macOS (Gatekeeper) o Windows (SmartScreen) van a avisar — click en "Abrir igual" / "Ejecutar de todas formas".
